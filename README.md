# Better-Dont-Disturb
Hello!
Android default dnd (Do not disturb) mode wasn't good enough so i made this
Better Don't Disturb (BDND) is a powerfull macrodroid macro that controles your volume, dnd mode and sound mode in the same macro, it has a lot of features like profiles, custom volume, volume saving, auto activation, etc.

# info
you can enter the wiki [here](https://github.com/felixgamer27/Better-Dont-Disturb/wiki)

# Download
To download the macro, go to the [versions folder](https://github.com/felixgamer27/Better-Dont-Disturb/tree/main/versions), select the version you want and download the macro using github or using the link to macrodroid store
